# APS1-Embarcados



### VIDEO: 
https://www.youtube.com/watch?v=aAyb5ndeDxc

### UTILIZANDO:

Botão 1 -> Toca a música.

Botão 2 -> Pausa a música.

Botão 3 -> Troca de música

OLED -> Indica qual música esta tocando

### CIRCUITO: 

Positivo Buzzer -> PD13

Negativo Buzzer -> GND

Placa Oled -> EXT1
